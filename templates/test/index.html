Basic Login Modal using Bootstrap and jQuery

A basic login modal with links to social media and the opportunity to login via email and password.

The Footer Modal
selecting wether to login or register

  • As a user, clicking on "sign-in" will launch the login modal.
  • As a user, clicking on "register" will loaunch the register modal.
  • As a user, clicking on the "" button will minimize the login/register footer.

The Login Modal

  • As a user, I can sign in via social media.
    • clicking on one of the buttons will launch the apps login page
  • As a user, I can choose to login via email/password. (via Angular, future version)
    • On "success", the modal will close and the user will be logged in.
    • On "error", the modal will stay up, and an error message will appear.
  • As a user, I can close the modal by clicking the "x".
  • As a user, I can choose to swith to the "register" modal.
    • The "login" modal will fade out and the "register" modal will fade in.

The Register modal

  • As a user, I can register via a social app.
    • Clicking on the button will launch the site's app to log in.
    • Upon successful registration, the modal will close and the user will be logged in.
    • Upon error, the error message will display and the user can try again.
  • As a user, I can register via email/ password/ confirmed password.
